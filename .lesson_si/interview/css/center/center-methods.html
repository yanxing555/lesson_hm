CSS居中方法大全

1. Flexbox居中

Flex
display: flex; justify-content: center; align-items: center;
✅ 现代浏览器支持好
✅ 代码简洁
❌ IE9不支持

2. Grid居中

Grid
display: grid; place-items: center;
✅ 一行代码解决
✅ 现代浏览器支持
❌ IE11部分支持

3. Absolute + Transform

Transform
position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
✅ 兼容性好
✅ 不需要知道尺寸
❌ 可能影响其他定位

4. Absolute + Calc

Calc
position: absolute; top: calc(50% - 20px); left: calc(50% - 40px);
✅ 精确控制
❌ 需要知道元素尺寸
❌ 计算复杂

5. Margin Auto

Margin
margin: 0 auto; /* 垂直居中需要额外处理 */
✅ 简单易懂
❌ 仅水平居中
❌ 需要设置宽度

6. Text Align

Text
text-align: center; line-height: 150px;
✅ 文本居中简单
❌ 仅适用于文本
❌ 垂直居中有限制

🎯 最佳实践推荐

  • 现代项目：使用 Flexbox 或 Grid
  • 兼容性要求高：使用 Absolute + Transform
  • 简单文本居中：使用 text-align: center
  • 响应式布局：使用 Flexbox 的 justify-content 和 align-items
