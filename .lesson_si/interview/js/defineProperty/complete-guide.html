🔧 属性描述符详解

属性 类型 默认值 说明
value 任意类型 undefined 属性值
writable boolean false 是否可修改属性值
enumerable boolean false 是否可枚举（for...in, Object.keys）
configurable boolean false 是否可删除或修改描述符
get function undefined 获取属性值时调用的函数
set function undefined 设置属性值时调用的函数
