# AI 单词拍照移动应用

- mobile App
- css reset
- 组件划分思维
  - 逻辑划分呢个图片上传
  - APP 根主键
  - PictureCard 子组件
  - 组件通信
    - 父组件负责持有我们的状态
    - 父组件api 请求
    - 子组件消费数据 
    - 父组件state(私有的数据状态) 子组件props(外界传入的数据状态) 都是数据
    - 子组件修改状态,通过回调函数通知父组件,父组件修改自己的状态
- 目录结构
  - src/ 开发目录
  - components/ 组件目录
    - 组件就是一个文件夹
    - jsx
    - css
  - public/ 静态资源目录
  - libs/ 工具包
  - .env.local  ai api-key token 等环境变量
